What is Zero Trust Network Access (ZTNA)

In this document, we're going to explore the basics of Zero Trust Networking and how can it be utilized to improve the overall security posture of an organization.

Definitionâ€‹

According to the research institute Gartner:

Zero trust network access (ZTNA) is a product or service that creates an identity- and context-based, logical access boundary around an application or set of applications. The applications are hidden from discovery, and access is restricted via a trust broker to a set of named entities.

ZTNA eliminates the implied authority given to authenticated users (i.e. just because you're connected to a network doesn't mean that you are authorized to access apps present on that network), thus enabling organization to tighten security controls against all access attempts, even authenticated users.

It's time to shift from Trust, but verify to Never trust, always verify.
